# ORA_CORE_OS_ORCHESTRATEUR_LLM_V2_0

Public technical bundle for `ORCHESTRATEUR_LLM`, the modular orchestration layer of `Ora_Core_Os`.

## Positioning

`ORCHESTRATEUR_LLM` does not replace `HGOV` or `GPV2`.

- `HGOV` governs truth, risk, memory and epistemic limits.
- `GPV2` carries structured semantic state for backend routing and stable transport.
- `ORCHESTRATEUR_LLM` decides which module to consult, in what order, and with what verification requirement.

Its purpose is to keep the permanent kernel short while routing each request toward the right technical module set.

## Core principle

`The kernel governs. The index guides. The modules specialize. Verification secures.`
