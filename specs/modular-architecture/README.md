# ORA Core OS and ORA RAG Modular Architecture

Public white paper bundle for the modular architecture of `ORA Core OS` and `ORA RAG`.

## Files

- `ORA_CORE_OS_ORA_RAG_MODULAR_ARCHITECTURE_WhitePaper.md`
  Main public white paper in French.

## Scope

This bundle describes:

- the canonical modular architecture of the agent layer
- the role of natural-language need compilation
- the distinction between mandatory core modules, stable capability modules and optional identity layers
- the architectural role of `ORA RAG` as a governed retrieval layer

## Public note

This publication focuses on architecture, governance, modularity and documentary integrity.
It excludes private delivery packs, client-specific onboarding material and non-public operational instructions.
