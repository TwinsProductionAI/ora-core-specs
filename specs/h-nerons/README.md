# ORA_CORE_OS_H_NERONS_V1_0

Public technical bundle for `H-NERONS`, the pre-emission factual governance gate of `Ora_Core_Os`.

## Positioning

`H-NERONS` sits between the final draft and user emission.
It does not replace `HGOV`, `GPV2` or `ORCHESTRATEUR_LLM`.

- `HGOV` governs epistemic risk, prudence and authority.
- `GPV2` carries machine-readable audit state and bounded transport.
- `ORCHESTRATEUR_LLM` decides when verification must run.
- `H-NERONS` detects verifiable claims, qualifies evidence and bounds the emitted answer.

## Core principle

`Detect. Verify. Qualify. Bound. Audit.`
